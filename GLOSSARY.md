# Translation glossary / 翻译术语表 / 翻訳用語集

This glossary standardizes recurring gameplay terminology for the **DMFD Witch** mod. It is intentionally not a complete list of cards, relics, powers, or events: exact content titles live in each `localization/<locale>/` directory and should be read directly from those JSON files.

本表用于统一 **DMFD Witch** 模组中反复出现的机制术语。它不会重复收录所有卡牌、遗物、能力和事件名称；这些内容在各语言中的准确标题以对应 `localization/<locale>/` 目录中的 JSON 为准。

この用語集は、**DMFD Witch** Modで繰り返し使われるゲーム用語を統一するためのものです。カード、レリック、パワー、イベントの名称一覧ではありません。各言語での正式な名称は、対応する `localization/<locale>/` ディレクトリのJSONを参照してください。

## Important distinctions / 易混淆项

- `释放 / Release` is the mechanic. The card titles are `法术释放 / Spell Release`, `精通释放 / Mastered Release`, and `龙注释放 / Dragon Release`.
- `释放 / Release` 是机制名称；三张牌的标题分别是“法术释放 / Spell Release”“精通释放 / Mastered Release”和“龙注释放 / Dragon Release”。
- 日本語では、仕組みの名称は「発動」、カード名は「魔法発動」「真・魔法発動」「ドラゴン発動」です。
- Use `Condense Sigil` for “凝聚法印”, but `Condensed ...` for generated card titles such as `Condensed Fire`.
- `Heal Sigil` is the Sigil name, while the generated card is `Healing`. Both are translated as “治疗” in Simplified Chinese.
- `Heal Sigil` 是法印名称，生成牌则叫 `Healing`；两者在简体中文中都使用“治疗”。
- 日本語では、`Heal Sigil` は「リカバリー刻印」、生成カード `Healing` は「リカバリー」です。

## Editing guidance / 编辑说明

- Preserve localization keys, SmartFormat placeholders, and gameplay markup exactly.
- 请完整保留本地化键、SmartFormat 占位符和游戏文本标记。
- ローカライズキー、SmartFormatプレースホルダー、ゲーム内マークアップは変更しないでください。
- When introducing or renaming a recurring mechanic, update this glossary together with every affected locale.
- 新增或重命名反复使用的机制术语时，应同时更新本表以及所有受影响的语言。
- Do not add every content title here. Card, relic, power, and event names remain authoritative in the locale JSON files.
- 不要在这里收录所有内容标题；卡牌、遗物、能力和事件名称仍以各语言 JSON 为准。
- すべてのコンテンツ名をこの用語集に追加しないでください。カード、レリック、パワー、イベントの名称は、各言語のJSONを正とします。
